
# IDRhythms v2

IDRhythms v2 reintroduces Matrix pencil, JTK-like, ARSER-like, and RAIN into the main workflow together with the previously available methods.

## Included methods
- Matrix pencil
- Cosinor
- Harmonic regression
- Lomb–Scargle
- FFT
- Wavelet
- Autocorrelation
- JTK-like
- ARSER-like
- Bayesian harmonic
- RAIN

## Run
```bash
cd IDRhythms_v2
./run.sh
```
